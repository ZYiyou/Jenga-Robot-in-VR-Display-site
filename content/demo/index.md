## Demo Walkthrough

1. Select a legal block using the VR controllers.
2. Robot arm approaches and grips the block in stages.
3. Extract and transport the block to the top placement.
4. Place the block at the top of the tower.
5. Tower stability is monitored, and collapse is resolved if necessary.
